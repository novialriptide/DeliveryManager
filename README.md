## How to play
1. You got x amount of time to make x amount of money!
2. Brown paths means __fragile__ packages will start to deteriorate!
3. Some packages can't go x amounts of distance!
4. Good luck!

## Build from Source
1. Install the [Python 3.10](https://python.org/) or newer
2. Run `pip install pygame==2.1.3.dev8`
3. Clone this repository
4. Type `python src/main.py`
